Bot Telegram RPG

🌟 Masuki dunia RPG waifu yang penuh aksi dan keimutan! Bot Telegram ini menghadirkan gameplay berbasis teks dengan fitur PvP waifu battle, explore dungeon, gacha summon, dan leaderboard ranking. Dibuat dengan cinta dan semangat waibu sejati~ 💘

✨ Fitur Utama

  • 🎮 PvP Turn-based Waifu Battle
  • 🎲 Gacha & Summon Sistem
  • 🏆 Leaderboard & Rank PvP
  • 🧭 Dungeon Eksplorasi
  • 🧠 Sistem Stat & Element

🛠️ Teknologi Digunakan

  • Telegraf.js & Telegram Bot API
  • Node.js (ESM)
  • MongoDB (Inventory, Stat, User)
  • Anime API / Custom JSON
